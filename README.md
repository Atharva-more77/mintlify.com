# Mintlify Landing Page Clone

A recreation of the Mintlify landing page - an intelligent knowledge platform for creating and maintaining world-class documentation.

## Project Overview

This project is a static HTML/CSS implementation of the Mintlify website landing page. It showcases Mintlify's key features including AI-powered documentation management, LLMs.txt & MCP integration, agent-based content management, and intelligent assistance for users.

## Features

- **Responsive Header** - Navigation bar with logo, links, and call-to-action buttons
- **Hero Section** - Eye-catching headline and email signup form
- **Feature Showcase** - Information about core Mintlify capabilities:
  - LLMs.txt & MCP integration
  - Self-updating knowledge management with Agent
  - Intelligent Assistant for users
- **Hero Images** - Visual demonstrations of platform capabilities

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or build tools required

### How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The page will display the Mintlify landing page

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Styling, flexbox layouts, and responsive design
- **Images** - PNG/SVG assets for visual elements

## Key Sections

1. **Header** - Fixed navigation with branding and actions
2. **Hero Content** - Main campaign message and signup call-to-action
3. **Feature Sections** - Detailed information about platform capabilities
4. **Visual Content** - Images demonstrating key features

## Browser Compatibility

This project is designed to work on all modern browsers that support:
- HTML5
- CSS3 Flexbox
- Modern CSS features

## Notes

- The page uses the Mintlify branding and design system
- Background images are loaded from external sources
- All links are placeholders and can be updated as needed
